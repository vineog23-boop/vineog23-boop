## Projetos em destaque

### [LocalTech](https://github.com/vineog23-boop/LocalTech)
API REST para locadora de veículos.  
Demonstra organização em camadas, modelagem de domínio e implementação de operações de back-end com Java.

### [gestao-vagas](https://github.com/vineog23-boop/gestao-vagas)
API REST para gestão de candidatos.  
Demonstra validação de entrada, persistência com Spring Data JPA e tratamento centralizado de exceções.

### [CadastroPersonagens](https://github.com/vineog23-boop/CadastroPersonagens)
Projeto de API REST voltado a CRUD e relacionamento entre entidades.  
Demonstra prática com Spring Boot, Flyway, mapeamento entre camadas e organização de código.

### [Portfolio-Java-Estudos](https://github.com/vineog23-boop/Portfolio-Java-Estudos)
Repositório curado de estudos e exercícios em Java.  
Funciona como base prática de fundamentos que sustentam os projetos back-end mais estruturados.
