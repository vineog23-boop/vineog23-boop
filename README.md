## Projetos em destaque

### [gestao-vagas](https://github.com/vineog23-boop/gestao-vagas)
API REST para gestão de candidatos.  
Demonstra organização em camadas, persistência com Spring Data JPA e tratamento centralizado de exceções.

### [locatech-api](https://github.com/vineog23-boop/locatech-api)
API REST para locadora de veículos.  
Demonstra modelagem de domínio, organização em camadas e implementação de operações de CRUD com Java.

### [ninja-missions-api](https://github.com/vineog23-boop/ninja-missions-api)
Projeto de API REST voltado a CRUD e relacionamento entre entidades.  
Demonstra prática com Spring Boot, Flyway, separação entre camadas e organização de código.

Repositório curado de estudos e exercícios em Java.  
Reúne fundamentos que sustentam a evolução para projetos back-end mais estruturados.
