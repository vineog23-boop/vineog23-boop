# Auditoria e Seleção de Repositórios

Esta auditoria foi realizada para identificar os projetos mais relevantes para o perfil de recrutamento back-end Java de **vineog23-boop**.

## Classificação de Relevância

| Repositório | Linguagem | Relevância | Justificativa |
| :--- | :--- | :--- | :--- |
| **Portfolio-Java-Estudos** | Java | **Alta** | Contém projetos práticos de POO (GameOfThronesPOO, HarryPotterPoo) que demonstram conceitos fundamentais como herança, interfaces e polimorfismo. |
| **Trabalho-Python-Academico** | Python | **Média** | Demonstra raciocínio computacional e capacidade de entrega acadêmica, embora não seja o foco principal (Java). |
| **projeto-travelgram** | HTML | **Baixa** | Projeto front-end básico, menos relevante para o foco em back-end Java. |
| **projeto-turismo** | HTML | **Baixa** | Projeto front-end básico. |
| **projeto-rockeatseat** | HTML | **Baixa** | Projeto front-end básico. |

## Seleção para Destaque (Pinned)

1.  **Portfolio-Java-Estudos**: O repositório principal de estudos Java, com foco em POO. É o "carro-chefe" do perfil atual.
2.  **Trabalho-Python-Academico**: Mostra versatilidade e base acadêmica sólida.

*Nota: Como o usuário possui poucos repositórios no momento, focaremos em extrair o máximo de valor do "Portfolio-Java-Estudos", transformando-o em um portfólio robusto.*
